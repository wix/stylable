Project Commons

The goal of the project file is to contain the CSS commons for the project, including variables, component variants and shared classes.

In the following code, you can see a project with:

  • 2 color and 2 font size variables
  • 1 Button component variant named cancelButton
  • 1 shared class for emphasisBox
/* project.st.css */
@namespace "Project";
:vars {
    color1: #F012BE;
    color2: #FF4136;
    fontBig: 30px;
    fontSmall: 10px;
}
:import {
    -st-from: './button/button.st.css';
    -st-default: Button;
}
.cancelButton {
    -st-extends: Button;
}
.emphasisBox {}

Note: We recommend calling the project commons file project.st.css.
